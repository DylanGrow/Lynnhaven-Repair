# Lynnhaven Small Engine Repair

A simple, fast, static website for displaying refurbished lawnmowers and small engine equipment.  
Built for GitHub Pages with no payment processing and no user accounts.

## Features

- Product listings
- Shopping cart using browser storage
- Order request form
- Mobile responsive layout
- Accessibility-friendly structure
- SEO-ready pages
- Local-only static site, easy to deploy

## Files

- `index.html` - Home page
- `shop.html` - Product catalog
- `cart.html` - Shopping cart
- `order.html` - Order request page
- `products.json` - Product data
- `style.css` - Site styling
- `script.js` - Cart and product logic
- `sitemap.xml` - Search engine sitemap

## Setup

1. Clone or download the repository.
2. Open `index.html` in a browser to test locally.
3. Update product data in `products.json`.
4. Edit contact information and branding in the HTML files.
5. Push the repository to GitHub.
6. Enable GitHub Pages in repository settings.

## GitHub Pages

In GitHub:
1. Go to **Settings**
2. Click **Pages**
3. Under **Build and deployment**, choose **Deploy from branch**
4. Select the `main` branch
5. Save

Your site will be published at:

`https://YOUR-USERNAME.github.io/YOUR-REPO-NAME/`

## Customization

- Replace placeholder product images with real photos.
- Update phone number, email, and location.
- Adjust product prices in `products.json`.
- Edit page titles and descriptions for local SEO.
- Add more products as needed.

## Notes

- No payment processing is included.
- No user accounts are included.
- Order submissions are handled as a basic request flow.
- The site is designed to stay simple and easy to maintain.
